# Manual do Usuário – Chama Já

## 1. Acesso como Cliente

1. Abra o aplicativo.
2. Informe seu nome e idade.
3. Selecione o perfil **Cliente**.
4. Digite a senha de 5 dígitos que o estabelecimento informou.
5. Se a senha tiver sido cadastrada pelo administrador, você será levado à tela de **Opções** (alarme, WhatsApp, Instagram).

## 2. Acesso como Administrador

1. Abra o aplicativo.
2. Informe seu nome e idade.
3. Selecione o perfil **Administrador**.
4. Digite a senha de administrador (definida no código como `87654321`).
5. Acesse a tela de **Gerenciar Senhas**.

### Gerenciar Senhas

- Para criar uma senha:
  - Digite o nome do cliente.
  - Toque em **“Gerar sequência (00001…)”**.
- Para editar:
  - Toque em **Editar** na senha desejada.
- Para apagar:
  - Toque em **Excluir**.
- Para enviar mensagem para o cliente:
  - Toque em **Opções** na senha desejada → você será redirecionado para a tela de **Opções** com o nome e a senha preenchidos.

## 3. Tela de Opções (Alarme, WhatsApp, Instagram)

- **Alarme**: envia uma notificação local com a mensagem “Sua senha é X”.
- **WhatsApp**:
  - Se informar um número (formato 55DDDNÚMERO), abre o WhatsApp direto com a mensagem pré-preenchida.
  - Se não informar número, abre o WhatsApp para você escolher o contato.
- **Instagram**:
  - Abre o Instagram Direct para o usuário configurado no código, com a mensagem “Sua senha é X” pré-preenchida.
